# FPLUS SEO — execution checklist

Print or tick weekly. Full guides: [docs/seo/README.md](../docs/seo/README.md).

## Week 1

- [ ] Complete [GBP setup](../docs/seo/01-google-business-profile-setup.md) — verify listing
- [ ] Upload 15+ photos to GBP
- [ ] Add all 8 services in GBP
- [ ] Copy [email signature](./email-signature.txt) to Outlook/Gmail

## Week 2

- [ ] Verify [Google Search Console](../docs/seo/02-search-console-setup.md) (DNS TXT)
- [ ] Deploy site (git push) — confirm robots.txt and sitemap.xml live
- [ ] Submit sitemap in GSC
- [ ] Request indexing: homepage + 5 service pages
- [ ] Run PageSpeed → log [week-00-baseline.csv](./tracking/week-00-baseline.csv)

## Week 3

- [ ] Submit 5 Tier-1 citations — [NAP guide](../docs/seo/03-nap-citations.md)
- [ ] Log in [citations-log.csv](./tracking/citations-log.csv)
- [ ] WhatsApp Business profile updated with fplus.ae

## Week 4

- [ ] Send 5 review requests — [scripts](../docs/seo/04-review-request-scripts.md)
- [ ] First GBP post
- [ ] Start [branded-queries.csv](./tracking/branded-queries.csv) from GSC

## Weeks 5–12

- [ ] 2 GBP posts per week
- [ ] 2 review requests per week (target 10 total by day 60)
- [ ] Monday: update tracking CSVs — [tracking guide](../docs/seo/05-branded-tracking.md)
- [ ] Check branded positions: `fplus ac repair`, `fplus hvac dubai`

## After deploy

- [ ] https://fplus.ae/robots.txt
- [ ] https://fplus.ae/sitemap.xml
- [ ] All service pages return 200
