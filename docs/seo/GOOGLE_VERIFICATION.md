# Google Search Console verification file

When Search Console offers **HTML file** verification:

1. Download the file (e.g. `googlea1b2c3d4e5f6.html`)
2. Place it in the **repository root** next to `index.html`
3. Push to GitHub Pages
4. Confirm: `https://fplus.ae/googlea1b2c3d4e5f6.html` loads in browser
5. Click **Verify** in Search Console

**Do not** commit a placeholder file — only add the exact file Google generates for your property.

Preferred method for `fplus.ae`: **DNS TXT record** at your domain registrar (see [02-search-console-setup.md](./02-search-console-setup.md)).
