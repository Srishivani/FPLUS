# Branded SEO & lead tracking — 60-day program

Track weekly for Phase 1. Use the CSV templates in [seo/tracking/](../../seo/tracking/).

## 1. Google Search Console (branded queries)

Every **Monday**:

1. Search Console → **Performance** → last 28 days
2. Filter **Country: United Arab Emirates**
3. Export or copy queries containing `fplus`
4. Paste into `seo/tracking/branded-queries.csv`

Columns: `week_ending`, `query`, `clicks`, `impressions`, `ctr`, `position`

### Targets by week 8

| Query (examples) | Target position |
|------------------|-----------------|
| fplus | 1–3 |
| fplus technical services | 1–3 |
| fplus ac repair | 1–3 |
| fplus hvac dubai | 1–5 |

## 2. Lead source tracking (WhatsApp & calls)

Ask every new lead: *"How did you find us?"*

Log in `seo/tracking/lead-sources.csv`:

| Source code | Meaning |
|-------------|---------|
| GBP | Google Business Profile / Maps |
| GORG | Organic Google search |
| GADS | Google Ads |
| WEB | Website direct |
| WA | WhatsApp referral |
| REF | Client referral |
| SOC | Instagram / Facebook |
| OTHER | Unknown |

### Weekly totals to record

- WhatsApp new conversations (WhatsApp Business → Analytics)
- Phone calls from GBP Insights
- Form/mailto enquiries from website
- Service page entries (after Phase 2)

## 3. GBP Insights (weekly)

From Google Business Profile → **Performance**:

- Search views vs Maps views
- Website clicks
- Call clicks
- Direction requests

Log in `seo/tracking/gbp-insights.csv`.

## 4. Review velocity

| Month | Target new Google reviews |
|-------|----------------------------|
| 1 | 5–10 |
| 2 | 4–8 |
| 3+ | 2–4 per month ongoing |

## 5. 60-day scorecard

At day 60, check:

- [ ] GSC shows branded queries with clicks
- [ ] GBP 100% complete, 10+ reviews
- [ ] 5+ Tier 1 citations live
- [ ] Lead log shows which channel converts best
- [ ] Ready to scale Phase 2 competitive keywords

## 6. UTM links (optional)

For GBP campaigns or posts, use:

```
https://fplus.ae/?utm_source=gbp&utm_medium=post&utm_campaign=ac_repair_may
```

Track in lead log under source **GBP**.
